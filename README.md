# 项目简介

## 核心功能
- 社团信息管理
- 社团活动管理
- 社团报名管理
- 留言板
- 社团会员管理
- 管理员操作

## 适用场景
- 高校社团内部管理
- 学生会日常事务处理
- 社团活动组织与报名
- 用户留言交流

## 技术特点
- 使用Django框架，保证系统稳定性和安全性
- 基于MySQL数据库，数据存储可靠
- 采用html、css、js进行前端开发，界面美观易用
